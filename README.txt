Task Management System

Overview

A web-based task management application that allows administrators to manage users and tasks, while enabling users to track and update their assigned tasks. The system includes email notifications for task assignments and a user-friendly interface.

Features
User Management (Admin only):

Add, edit, and delete users

Assign roles (Admin/Regular User)

Task Management:

Create, assign, and track tasks

Set deadlines and priorities

Update task status (Pending, In Progress, Completed)

User Dashboard:

View assigned tasks

Update task status

Change password

Notifications:

Email alerts for new task assignments


System Requirements

PHP 7.4 or higher
MySQL 5.7 or higher
Web server (Apache recommended)
SMTP server for email functionality

Installation Instructions (Using XAMPP)

Install XAMPP:

Download and install XAMPP from https://www.apachefriends.org

Start Apache and MySQL services from the XAMPP Control Panel

Set Up Database:

Access phpMyAdmin at http://localhost/phpmyadmin

Create a new database named "db_tasksystem"

Import the provided SQL file (if available) or run the database creation script

Accessing the System

Open your web browser and navigate to:
